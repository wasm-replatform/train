14) [mechanical] If no previous state string initialize dilaxState with default zeros/nulls.
15) [domain] Perform backward-compatibility migration: read migrated flag key; if absent read tripIdOldVersion and countOldVersion from legacy keys; if present set dilaxState.lastTripId and dilaxState.count accordingly; set migrated flag.
16) [mechanical] Else parse previous state JSON into dilaxState.
17) [mechanical] Parse token from dilaxEvent.clock.utc as integer.
18) [domain] If token <= dilaxState.token log warn and skip updating counts (duplicate message condition).
19) [domain] Else set dilaxState.token to token and evaluate tripId change logic.
20) [domain] If tripId present and dilaxState.lastTripId null set lastTripId to current tripId (prevent reset on first run).
21) [domain] If lastTripId exists and differs from current tripId set hasTripIdChanged true and update lastTripId.
22) [domain] If no tripId OR hasTripIdChanged then reset dilaxState.count to 0 and call updateRunningCount with skipOut=true else call updateRunningCount skipOut=false.

24) [infrastructure] Write updated dilaxState to Redis using getset; set expiry Config.APC_TTL_SECS; compare lastValue with previous string to detect dirty write; log warn if mismatch.
25) [infrastructure] Backward-compatibility: If occupancyStatus set write to legacy keyOccupancy with TTL REDIS_TTL; catch errors report to newrelic and log.
26) [infrastructure] Backward-compatibility: Write passenger count to legacy apcVehicleIdKey with TTL Config.APC_TTL_SECS; catch errors similarly.

28) [domain] After lock call saveVehicleTripInfo(vehicleId, vehicleLabel, tripId, stopId, event).
29) [mechanical] Return dilaxEventEnriched.
30) [domain] updateRunningCount: Iterate each door in dilaxEvent.doors summing in and conditionally out; compute currentCount = max(previousCount - outTotal,0)+inTotal; set dilaxState.count; log.
31) [domain] updateOccupancyStatus: Compare dilaxState.count against seating and total thresholds (5%, 40%, 90% seating; 90% total) assign OccupancyStatus enum or FULL; store string form.
